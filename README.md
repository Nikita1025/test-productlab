## Приложения для просмотра картинок 


####  Стек технологий:
* React
* TypeScript
* Redux Toolkit Query
* React Hook Form
* React router dom
* Zod
* Material ui / Radix UI / SCSS
* MSW - для эмитации запросов на сервер



####  Чтобы запустить проект:
1. npm install --global yarn
2. yarn
3. yarn start
